Backend - Controle de Estoque

API REST desenvolvida com Spring Boot para gerenciamento de produtos.

## 🔗 Projeto completo
- Frontend: https://github.com/badovb/estoque-front

  ### 🌐 Acesse o sistema
https://controle-estoque-api-esnm.onrender.com/produtos

### 📡 Endpoints

- GET /produtos
- POST /produtos
- PUT /produtos/{id}
- DELETE /produtos/{id}
- GET /produtos/buscar?nome=...

## 🛠 Tecnologias
- Java
- Spring Boot
- PostgreSQL

## ▶️ Como rodar
- Configurar banco PostgreSQL
- Rodar aplicação
